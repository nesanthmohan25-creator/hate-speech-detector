# Hate Speech Detector

A machine learning web application that detects and classifies hate speech in text using BERT and Streamlit.

##  What It Does
- Classifies input text as hate speech or clean
- Highlights profanity and harmful words
- Extracts flagged terms for analysis
- Simple web interface built with Streamlit

##  Tech Stack
- Python
- BERT (Hugging Face Transformers)
- Streamlit
- Pandas, NumPy

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
